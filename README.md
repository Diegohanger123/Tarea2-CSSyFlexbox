# Tarea2-CSSyFlexbox

- Enlace a github pages: https://github.com/Diegohanger123/Tarea2-CSSyFlexbox

- 3 propiedades de CSS que utilice:
  - flex-direction: row;
    Utilizando Flexbox, hace que sus elementos se organicen en fila
  - justify-content: center;
    Centra los elementos en el Flexbox, en este caso que estan en fila se centran horizontalmente, si fuera en columnas se haria verticalmente.
  - box-shadow: 0px 5px 10px rgb(0, 0, 0);
    Agrega una sombra debajo de un elemento especificando su tamaño y el blur que tiene ademas de el color.
